# 个人网站

这是一个使用HTML、CSS和JavaScript构建的个人网站，采用暗色主题+neon风格设计。

## 页面结构

- 首页
- 关于我
- 作品集
- 联系方式

## 设计特色

- 暗色主题配色方案
- neon风格的视觉效果 (#00ffea 主色)
- 响应式设计
- 现代化的用户界面

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)
- 无框架纯JavaScript实现

## 部署

网站通过GitHub Actions自动部署到GitHub Pages。

## 本地开发

```bash
# 克隆仓库
git clone <仓库地址>

# 启动本地服务器
python3 -m http.server 8000

# 访问网站
打开浏览器访问 http://localhost:8000
```

## 功能特性

- 作品集分类筛选
- 表单验证和提交
- 社交媒体链接集成
- 完整的导航系统
- 霓虹灯动画效果
